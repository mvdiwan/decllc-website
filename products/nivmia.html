Roadmap

Key Features

Everything you need to monitor, manage, and analyze your network infrastructure.

Multi-Vendor Collection

SSH, SNMP, telnet, and API access methods across all 17 platforms. Run any single method or any combination. Method failures never block other methods from completing.

Configuration Vault

Git-backed configuration versioning with golden copy management. Full diff history, tag known-good configs, and track every change with auditable commits.

Monitor vs Manage

Capability-based licensing. Community tier provides read-only monitoring. Professional and Enterprise tiers unlock full configuration management with pre-check, apply, post-check, and rollback.

AI-Powered Analysis

On-premises AI analysis powered by local Ollama models. Device analysis, config anomaly detection, and advisory recommendations. Your data never leaves your network.

Pluggable Architecture

Per-platform RPM and DEB packages. Install only the vendor plugins you need. Each plugin is self-contained with its own orchestrator, collectors, and distributors.

Prometheus + Grafana

Built-in metrics exporter with 12 metrics: device up/down, CPU, memory, temperature, uptime, interface count, and collection stats. Pre-built Grafana dashboards included.

Architecture

Pluggable, configuration-driven, platform-isolated by design.
