Key Capabilities

Production-grade VM backup without the enterprise price tag. Replaces XSIBackup and proprietary backup agents.

ESXi VM Backup

Stage ESXi virtual machine backups using native vmkfstools for thin-provisioned, consistent snapshots. No agents needed inside the VM.

KVM/Proxmox Integration

Orchestrates vzdump for Proxmox and virsh snapshot for KVM/libvirt. One tool for all your hypervisors.

CBT-Aware Incrementals

Changed Block Tracking support for ESXi means incremental backups transfer only what changed. Dramatically reduces backup windows and storage.

Cross-Site Replication

Rsync-based replication to local storage, remote sites, or cloud targets. Works over VLAN trunks, VPNs, or WAN links.

Scheduled Retention

Configurable retention policies — daily, weekly, monthly. Automatic pruning of old backups. GFS (Grandfather-Father-Son) rotation support.

Disk-Level Intelligence

Skip non-essential disks (syslog archives, temp data) with skip_disks configuration. Exclude VMs by tag or pattern.
